**node 단일 실행 ( ros2 run )**

```python
ros2 run <PKG Name> <Node Name>

# ros2 run turtlesim turtlesim_node
```


**실행중인 node 확인 ( ros2 node list )**

```python
ros2 node list

# 실행결과 : /turtlesim
```


**노드 정보 ( ros2 node info )**

```python
ros2 node info /turtlesim
```
